This repository contains the Full Stack Car Dealership Application, developed as part of the Full Stack Application Development Capstone Project.  
It integrates Django, React, Express, and MongoDB with CI/CD and Containerized Deployment.

## Features
- Django backend for static pages, user management, and proxy APIs
- Express + MongoDB backend for dealers & reviews
- React frontend for UI (login, signup, dealers, reviews)
- CI/CD via GitHub Actions
- Dockerized deployment

## How to run (local dev)
1. Clone:
   git clone https://github.com/VaishnaviKadam07/dealership-app.git
2. Follow each component's README (djangoapp/, server/, frontend-react/)
